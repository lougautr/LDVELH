Projet de Livre dont vous êtes le héros

C'est une progression de lecture en fonction des choix et du résultat des actions du lecteur.
C'est un editeur de texte, dans lequel le joueur peut créer sa propre histoire dont le lecteur est le héros.

Cet editeur se présenter sous forme de page vierge avec une fenetre qui demande à l'utilisateur de taper l'introduction de son histoire, puis les différents chapitres, qui conduisent à d'autres chapitres en fonction des choix choisis. D'un autre côté, le joueur peut choisir de charger son histoire précédemment écrite, ou une déjà existante. Une fois l'histoire chargée, le joueur peut jouer de manière basique en suivant les choix déjà définis, ou d'en créer, ou supprimer en temps réel. Un fois la fenêtre du jeu fermée, le graphique de l'histoire s'affiche sous format .png.

Prérequis:
	Graphivtz disponible à l'adresse suivante: http://www.graphviz.org/

Déroulement:
	L'application s'éxecute grâce à ANT. Se placer dans scripts/.
		- exécuter "ant init" pour télécharger les librairies nécessaires (à faire une seul fois)
		- exécuter "ant run" pour démarrer le logiciel
		- exécuter "ant delete" pour supprimer les libraires
